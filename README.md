# 🎵 Расширение для скачивания музыки с Яндекс.Музыки

Простое и удобное расширение для браузера, позволяющее скачивать любимые треки напрямую с [Яндекс.Музыки](https://music.yandex.ru).

## 📥 Установка из исходников

Поскольку расширение не опубликовано в официальных магазинах, его необходимо установить вручную в режиме разработчика.

1. **Скачайте** и распакуйте на свой компьютер.

2. Откройте страницу расширений в вашем браузере:
   - **Chrome:** `chrome://extensions`
   - **Edge:** `edge://extensions`
   - **Opera:** `opera://extensions`
3. Включите **«Режим разработчика»** (обычно переключатель находится в правом верхнем углу страницы).
4. Нажмите кнопку **«Загрузить распакованное расширение»**.
5. Выберите папку с файлами этого расширения.

## 🔐 Авторизация (Важно!)

Перед началом использования необходимо выполнить первичную настройку:

1. После установки расширения зайдите в свой аккаунт Яндекса.
2. Вам потребуется пройти авторизацию через **OAuth**. Следуйте инструкциям на экране для предоставления доступа.
3. После успешной авторизации перейдите на страницу [music.yandex.ru](https://music.yandex.ru).

### Способы скачивания:

*   **Плавающая кнопка:** Справа снизу появится желтая кнопка ⬇️. Вы можете перетащить её в любое удобное место экрана. Нажмите на неё, чтобы скачать текущий играющий трек.
*   **В списке треков:** Рядом с любым треком в плейлисте или поисковой выдаче появится кнопка ⬇️ для быстрой загрузки.
*   **Альбомы и плейлисты:** На страницах альбомов или плейлистов появится общая иконка «Скачать» для массовой загрузки.

## ⚠️ Возможные проблемы

*   **При нажатии на скачивание ничего не происходит:**
    *   Попробуйте **обновить страницу** (F5).
    *   Иногда требуется немного **подождать**, так как получение ссылки на файл может занять время.
    *   Убедитесь, что вы успешно прошли OAuth-авторизацию.
